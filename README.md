## «Rss Aggregator»
[Service for receiving and displaying RSS feeds](https://frontend-project-11-hgf5.onrender.com/)


It allows you to add an unlimited number of RSS feeds, updates them every 5 seconds and displays new entries.
---

### Setup

- clone the repository 

``` 
$ git clone <url>
```

- move to directory project 

```
$ cd frontend-project-11
```

- install the package and all its dependencies

```sh
$ make install
```

- run on localhost

```
make develop
```

### Minimum requirements

Project "Rss Aggregator" requires [Node.js](https://nodejs.org/) (Latest LTS Version: 20.13.1 (includes npm 10.5.2))
